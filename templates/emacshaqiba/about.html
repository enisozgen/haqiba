About Haqiba

This page is about Haqiba. The site provide easy to use Emacs config file(s) which can be customized. The config file is ready to use with Emacs. The configuration is tested on GNU Emacs version 24.3. It may work on older versions but I cannot assure you that.

If you are new to GNU Emacs then let me tell you that it is a powerful and easy to use text editor. It is highly customizable and extensible editor. If you have not chosen your editor yet then give GNU Emacs a try and you will end up using Emacs your entire life. The basic text editing is similar to any other text editor. As a new user you can certainly use a menu bar to perform daily tasks like cut, copy, paste, and save. Once you get familiar with Emacs, start using key-shortcuts and more advance features.

GNU Emacs has been around since 1970's and it was originally written by Richard Stallman and Guy L. Steele, Jr. Its an interpreter to Emacs Lisp, a dialect of the Lisp programming language.
